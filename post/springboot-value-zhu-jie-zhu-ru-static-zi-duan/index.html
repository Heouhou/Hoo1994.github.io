SpringBoot @Value 注解注入static 字段

2019-05-14

普通参数使用@Value 示例:

public class Utils {
    @Value("${test.host}")
    private  String host;

    @Value("${test.port}")
    private  String port;

    ......
}

但是在注入static字段时,直接使用 @Value 为静态变量赋值是不行的，可以使用 set 方法：

@Component
public class Utils {
    private static String host;

    @Value("${test.host}")
    public void setHost(String host) {
        Utils.host = host;
    }

    private static String port;

    @Value("${test.port}")
    public void setPort(String port) {
        Utils.port = port;
    }

    ......
}

注意，需要使用 @Component 注解。
